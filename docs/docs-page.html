A bit about V++

V++ is a small static-typed, compact, interpreted language made for running small programs in a console window. It is greatly documented and easy to use.

A small example

0.1. Downloading and running V++

0.1.1. Downloading V++

In order to download V++ click here.
Download the lastest version for the best performance, less bugs and more functionality.

0.1.2. Installing V++

Once you downloaded the V++ setup you can start it.
When the window pops up click "Next" and then the setup will continue to the next page. On the second page it is recommended to leave all the settings as they are, after that click "Next" once more. Wait a few seconds and then... V++ is installed!

0.2. Installing VSCode and the VSCode V++ extension

0.2.1. Installing VSCode

VSCode (Visual Studio Code) is a text-editor made by Microsoft. In order to download it click here.

0.2.1. Installing the VSCode V++ extension

Go to the Extensions button in VSCode an then search V++ in the "Search Extensions in Marketplace" searchbar. Install the first extension.

1. Basic types and setup

1.1. Types

There are currently 5 types in the V++ programming language. (4 dynamic types and 1 static type) Dynamic types represent types which are defined at runtime. Static types represent types which are defined at script setup.

Type Name Static/Dynamic Is Complex? Example Notes/Use Version added
function Static true Click here A block of code. 1.0.0.0
number Dynamic false 1234.5678 Can store a number. 1.0.0.0
string Dynamic false "Hello world!" Can store text. 1.0.0.0
bool Dynamic false true Can store a value of true or false. 1.0.0.0
list Dynamic true - Can hold multiple values. 1.1.1.7
